# UAS-Grafika-Komputer-B

Membuat sebuah simulasi dimana seorang "player" dapat menjelajahi sebagian kecil lautan yang diisi dengan berbagai poly berbentuk hewan, terumbu karang, dan berbagai hal lainnya yang biasanya dapat ditemukan dalam laut.
Proyek akan menggunakan ThreeJS 3D dan jika waktu cukup, akan ditambahkan bagian atas laut yang berisikan langit dan samudera untuk pengalaman yang lebih in-depth.
